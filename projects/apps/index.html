G-Licious!

Role: UX Research & iOS7 Redesign Lead

G-Licious is a quick and easy way to check the Dining Menu for the Grinnell College Marketplace. You can check the menu for today, tomorrow, or any day in the next week as and when a new menu becomes available. You can also filter out choices by personal dietary concerns.

Developed by Grinnell AppDev, specifically @DrJid & @tremblay.

The main focus of the redesign was based on prototyping and wireframing multiple designs. Through the Facebook group I've created for the Grinnell Appdev student group I called out for participants that wanted to take part in redesigning our most downloaded app - G-Licious.

43 students participated in the study, and together we were able to decide how best to proceed. After 3 iterations, and the data gathered through them, we were able to come up with a more intuitive user-interaction scheme that fit the iOS7 interface, and built an app that was much more self-evident to the user. Grinnell Events!

Role: iOS Developer, UX Designer, Graphics Designer

Grinnell Events lets users see all events happening on the college campus, and add them to their calendar.

The main view presents the user with an overview of all events sorted by day. Each event has attributes stating their name, location, and start and end time. By clicking on an event a user can see a more detailed description of the event, as well as add the event to their calendar. An graphical indicator lets the user know if they're available, unavailable, or are already attending a certain event.

The user can specify what calendar they want to add the event to by going to the settings page. If a user wants to find a specific event, the main view provides a search function by title.

Built with the help of Grinnell AppDev, specifically @DrJid.

Scripts

Role: iOS Developer, UX Designer, Graphics Designer

Scripts is a crowdsourced storytelling application that enables users to create and contribute to stories that are shared all across the globe.

Each user will be able to watch his or her story grow as it travels around the world and connect others who have taken an interest in her story. Users will be required to contribute to a number of preexisting stories before they will be able to start their own, however they will have a variety of stories to choose from which come along with summaries of the original authors vision for the plot.

Built at MHacks Fall 2013 Hackathon with @DrJid

Grinnell S&B!

Role: iOS Developer, UX Designer, Graphics Designer

S&B is a mobile news reader for Grinnell College's most prominent weekly publication - the Scarlet & Black.

The application currently lets you see all the news starting with the most recent ones. A user can also specify looking at news according to one of the 4 other implemented categories - Sports, Community, Opinion, and Arts. Each short article description leads you to the full article where any related photos, and authors are displayed.

Currently working on: updating for iOS, integrating comments & sharing through social media

Built with the help of Grinnell AppDev, specifically @tremblay.
